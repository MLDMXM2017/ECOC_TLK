# ECOC_library

The repository offers several impelment of ECOC algorithms, including    
  - OVA ECOC  
  - OVO ECOC   
  - Dense random ECOC
  - Sparse random ECOC
  - D ECOC
  - AGG ECOC
  - CL_ECOC
  - ECOC_ONE
  - ECOC_MDC
  
The last one is proposed by me, so if anyone want to know more detail, please contact with me.
